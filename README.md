# Association Finance Simulator

協会・団体向けの収支予測シミュレーターです。

過去3〜5年分の会員数、収入、支出データを入力すると、将来の収支を簡易的に試算できます。  
物価上昇、人件費上昇、印刷・郵送費上昇を分けて設定し、「何もしなかった場合」と「介入した場合」を比較できます。

## 主な機能

- 過去3〜5年分の収支データ入力
- 物価・人件費・印刷郵送費を分けた将来試算
- 無介入シナリオと介入シナリオの比較
- 赤字化年の表示
- 必要会費の目安表示
- グラフ表示
- 入力内容のブラウザ内保存
- 試算結果のCSV出力

## 使い方

1. `index.html` をブラウザで開く
2. 過去の会員数・収入・支出を入力する
3. 将来の上昇率を設定する
4. 介入シナリオを設定する
5. 「収支予測を計算する」を押す

## 注意

本ツールは、将来を正確に予測するものではありません。  
前提条件を変えたときに、協会財政がどのように変化するかを確認するための簡易シミュレーターです。

実際の予算編成や会議資料として使用する場合は、決算書・予算書・会計担当者による確認と併用してください。
